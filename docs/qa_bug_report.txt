1. BROKEN RESOURCE LINKS
--------------------------------------------------------------------------------
* Location:         "A roadmap for learning cybersecurity" page (/learning-roadmap)
* Section:          Under "Resources" section
* Issue:            Government, NIST, and TryHackMe hyperlinks fail to load (return 404 or dead URLs).
* Steps to Repo:    1. Navigate to the learning roadmap page.
                    2. Scroll down to the "Resources" section.
                    3. Click on the NIST or TryHackMe external links.
* Expected Result:  User is redirected to the correct external resource page in a new tab.
* Actual Result:    The browser attempts to load a broken URL or throws a 404 error page.
* Impact:           Users cannot access external study materials.


2. UI TEXT VISIBILITY / TIMELINE TRUNCATION
--------------------------------------------------------------------------------
* Location:         "A roadmap for learning cybersecurity" page (/learning-roadmap)
* Section:          Step-by-step roadmap sections / timeline components
* Issue:            Roadmap sequence numbers are invisible, cut off, or hidden behind other layout elements.
* Steps to Repo:    1. Open the learning roadmap page.
                    2. Scroll through the chronological roadmap timeline steps.
                    3. Observe the step numbers (1, 2, 3, etc.).
* Expected Result:  Sequence numbers should be clearly visible, fully rendered, and aligned with the text.
* Actual Result:    Numbers are severely truncated, cropped, or completely hidden due to layout constraints.
* Impact:           Users cannot see the proper learning order.

--------------------------------------------------------------------------------
* Location:         Explore Threat Intelligence dashboard (/threat-intel-dashboard)
* Section:          Inside individual "Intelligence Reports" feed
* Issue:            Clicking the "View Report" buttons registers no action (missing event listener or broken route).
* Steps to Repo:    1. Navigate to the Explore Threat Intelligence dashboard.
                    2. Locate any individual "Intelligence Report" card.

* Section:          Whitepapers and article listings

                    2. Locate any listed whitepaper or article.

* Expected Result:  A local file download sequence is successfully triggered, saving the PDF asset.
* Actual Result:    Nothing downloads; the system console may throw a file asset path error.
* Impact:           Users cannot download or access offline research documents.
